About


Geoinformatics is the science and the technology which develops and uses information science infrastructure. Geoinformation science plays a role in all domains and disciplines making use of geospatial information, such as:

  • Resource Management
  • Risk and Disaster Management
  • Navigation systems
  • Logistics
  • Crime analysis

Geoinformatics uses geocomputation and geovisualization for analyzing geoinformation in order to solve problems. It's disciplines range from Remote Sensing over Photogrammetry to Spatial Analysis. Making use of:

  • Computer vision
  • Machine learning
  • Big Data techniques
  • Social Network Data Analytics
  • Data mining

Big Data

With the rapid development of computational as well as IT infrastructure, the data to analyze also grew. In order to deal with vasts amount of data, a solution is out of core processing.

The NY Taxi data from 2015 has a size of 22.9 GB. Even though, some would not considere that Big Data, however using Dask, this data can be analyzed on a Desktop with only 16gb ram using simple Python. When switching to Software explicity developed for Big Data such as Hadoop, workflows have to be changed.

  • Library: Dask
  • Data: NY Taxi (2015)
